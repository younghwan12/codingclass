1부터 100까지 출력하기

1.1 while문을 이용해서 1부터 100까지 출력하시오!

{
    let num = 1;

    while(num <= 100){
        document.write(num);
        num++;
    }
}
결과보기
123456789...979899100

1.2 for문을 이용해서 1부터 100까지 출력하시오!

{
    for(let i = 1; i<=100; i++){
        document.write(i);
    }
}
결과보기
123456789...979899100

1.3 for문을 이용해서 1부터 100까지 출력하시오(짝수, 홀수)

//짝수
{
    for(let i = 2; i<=100; i+=2){
        document.write(i);
    }
}
//홀수
{
    for(let i = 1; i<=100; i+=2){
        document.write(i);
    }
}
