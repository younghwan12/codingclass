배열 관련 메서드

자바스크립트 배열 관련 메서드중 오늘은 join(), push(), pop()에 대해서 소개드리려고 합니다.

join()

배열 요소 문자열을 결합할 수 있고, 문자열을 반환할 수 있습니다. 자주쓰이며 중요합니다! 예시로 설명 드리겠습니다. 


// 예시입니다.
const arrNum = [100, 200, 300, 400, 500];

const text1 = arrNum.join('');  //리턴값이 공백 없이 나옵니다.
const text2 = arrNum.join(' '); //리턴값이 공백을 추가해서 나옵니다.
const text3 = arrNum.join('★'); //리턴값에 ★을 추가해서 나옵니다.
const text4 = arrNum.join('-'); //리턴값에 -을 추가해서 나옵니다.
결과 확인하기
100200300400500
100,200,300,400,500
100★200★300★400★500
100-200-300-400-500

push()

push() 메서드는 배열 마지막 요소에 추가 하고, pop() 메서드는 배열 마지막 요소를 삭제합니다. 예시로 설명드리겠습니다. 


// 예시입니다.
const arrNum = [100, 200, 300, 400, 500];
const arrPush = arrNum.push(600);

document.write(arrNum);     //리턴값은 배열의 갯수를
document.write(arrPush);    //100,200,300,400,500,600 결과값은 전체 출력
결과 확인하기
6
100,200,300,400,500,600

pop()

배열의 아이템을 삭제할 수 있습니다. push메서드와 마찬가지로 배열의 끝을 삭제합니다. 예시로 설명드리겠습니다. 


// 예시입니다.
const arrNum2 = [100, 200, 300, 400, 500];
const arrPush2 = arrNum2.pop();

document.write(arrPush2) //500 리턴값은 마지막요소를
document.write(arrNum2) //100,200,300,400 결과값은 리턴값을 제외한 나머지를 출력
결과 확인하기
500
100,200,300,400
표
메서드 설명
join() 배열의 아이템을 합칠 수 있다.이때 각 아이템의 구분자를 지정할수있다.
(ex: join(''),,)
push() 배열에 아이템을 밀어 넣을 수 있다.
(배열의 끝에 추가)
pop() 배열에 아이템을 삭제할 수 있다.
(push메서드와 마찬가지로 배열의 끝을 삭제)
