펼침 연산자

간단히 말해서, 배열에 포함된 항목을 목록으로 바꿔주는 연산자입니다.
마침표 세 개(...)로 표시합니다.
펼침 연산자는 단독으로 쓰일 수 없음에 주의해야 합니다.
배열에 포함된 항목을 목록으로 바꿨다면 이를 배열이나 객체 등에 담아줘야 한다. 변수에 담게 되면 에러가 발생합니다.

펼침 연산자 - 복사

const obj = {
    a: 100,
    b: 200,
    c: "javascript"
}

const spread = { ...obj };
document.write(spread.a);
document.write(spread.b);
document.write(spread.c);
