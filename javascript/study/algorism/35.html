프로그래머스 1단계 : H-Index

코딩테스트 연습 > 정렬 > H-Index

✔ 문제 설명

H-Index는 과학자의 생산성과 영향력을 나타내는 지표입니다. 어느 과학자의 H-Index를 나타내는 값인 h를 구하려고 합니다. 위키백과1에 따르면, H-Index는 다음과 같이 구합니다. 어떤 과학자가 발표한 논문 n편 중, h번 이상 인용된 논문이 h편 이상이고 나머지 논문이 h번 이하 인용되었다면 h의 최댓값이 이 과학자의 H-Index입니다. 어떤 과학자가 발표한 논문의 인용 횟수를 담은 배열 citations가 매개변수로 주어질 때, 이 과학자의 H-Index를 return 하도록 solution 함수를 작성해주세요.

✔ 제한 사항

과학자가 발표한 논문의 수는 1편 이상 1,000편 이하입니다.
논문별 인용 횟수는 0회 이상 10,000회 이하입니다.

✔ 입출력 예

citations return
[3, 0, 6, 1, 5] 3

✔ 문제풀이

function solution(citations) {    
    citations = citations.sort((a, b) => b - a);
    let answer = 0;
    while (answer + 1 <= citations[answer]) {
        answer++;
    }    
    return answer;
}

✔ 설명

논문 인용횟수를 내림차순으로 정렬하고 배열 길이만큼 while문을 돌리면 H-index가 나오는 시점에 반복문이 멈춘다. 그때 출력하면 끝 문제를 이해하기가 어려운 문제였습니다.

✔ 다른사람의 풀이

function solution(citations) {
    citations = citations.sort(sorting);
    var i = 0;
    while(i + 1 <= citations[i]){
        i++;
    }
    return i;


    function sorting(a, b){
        return b - a;
    }
}

다들 비슷하게 풀었네요 이방법은 정렬을 마지막에 해줬습니다.
