font-size, 단위의 고찰

1. px

해상도에 따라 상대적으로 달라지는 기본 단위로서, 다음과 같이 표현합니다

font-size: 12px;

2. %

부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위로서, 다음과 같이 표현합니다.

font-size: 150%;

특별한 설정이 없다면 16px이 100%가 되므로, 150%는 24px이 됩니다. 만약 부모 요소가 10px이었다면 150%는 15px이 될 것입니다.

뷰포트는 모바일 해상도(가로보기, 세로보기)에 맞게 HTML 문서를 조절해주는 속성이라고 생각하면 된다.

3. em

부모 요소의 글자 크기를 100% 기준으로 계산한 100 분의1 단위로서, 다음과 같이 표현합니다.

font-size: 1.5em;

특별한 설정이 없다면 16px이 1em이 되므로, 1.5em은 24px(16px X 1.5 = 24px)이 됩니다. 만약 부모 요소가 10px이었다면 1.5em은 15px이 될것입니다.
